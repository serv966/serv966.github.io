Caution before the Adjustment/Confirmation

  • Before starting any adjustment, read and observe all safety precautions shown in the primary service manual
  • Make sure that all PCBs, chassis parts and connectors are in the right positions.
  • Assure the AC input power source is within the specified operating range, ± 2volts.
  • Receive any signal and perform the pre-heat more than 30 minutes under the normal temperature.

  • Please return the User Time to "0 hours" by using the rewriting software when the repair mentioned below is performed.          
    • When replacing the CRT.
  • When the parts mentioned below are exchanged, the User Time is "0 hours". By using the rewriting software, please return the User Time at the time when the set is returned.
    • When exchanging the EEPROM for microcomputer (U353).
    • When exchanging the PCB(ASSY PCB-MAIN) which has a EEPROM for microcomputer.

Do not short any position of the circuit while the monitor is in operation.

This will cause smoke, electric shock or damage to transistors, ICs or other parts or circuit in the unit. (Excluding the adjustment only when specified).

Listed signals on " Signal Timing Chart" are required. If they are not available, use properly programmed signal generator with signal timing of the above mentioned signal source.
