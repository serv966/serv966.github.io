Tools

Before starting the operation test, the following tools are recommended to do below tuning :

  • Hub Tester
  • USB Cable x 5
  • USB Compliant PC (Host PC)
    • Intel82371** PCI to USB Universal Host Controller can be found at Control Panel \ System \ Device Manager \ Universal Serial Bus Controller
  • Hub Tester Program Ver.2.10
    Install the below Hub test programs into the USB compliant PC.

For all CRT displays, L671, L771 and i.Staion

  • b.bat (Test program for over-current protection test: current supplied 0.5A × 4 ports = 2A)
  • a.bat (Test program for except over-current protection test)
 usb_uht.exe
 USB_uht.cfg


* The 1 current is supplied to 4 ports at the same time.
