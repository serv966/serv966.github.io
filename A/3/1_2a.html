Factory Model Menus

Size Menu
Size
With the right/left arrow key, the horizontal display size gets adjustable. After the adjustment, the size adjustment is over by pressing enter-key without adjustment data save.
Position
With the right/left arrow key, the horizontal position gets adjustable. After the adjustment, the position adjustment is over by pressing enter-key without adjustment data save.
Side Pin
With the right/left arrow key, the Side Pin Balance gets adjustable. With top/bottom arrow key, the Side pin distortion and pin-cushion distortion gets adjustable. After the adjustment, the side pin distortion adjustment is over by pressing enter-key without adjustment data save.
Trapezoidal
With the right/left arrow key, the Trapezoidal distortion gets adjustable. After the adjustment, the Trapezoidal distortion adjustment is over by pressing enter-key without adjustment data save.
Side Pin.C-Top
The Side Pin. Corner Top gets adjustable with the right/left arrow key. After the adjustment, the Side Pin. Corner Top adjustment is over by pressing enter-key without saving adjustment data.
Side Pin.C-Btm
The Side Pin. Corner Bottom gets adjustable with the right/left arrow key. After the adjustment, the Side Pin. Corner Bottom adjustment is over by pressing enter-key without saving adjustment data saving.
Side Pin.S/W
With the right/left arrow key, the W type correction gets adjustable. With the up/down arrow key, the S type correction gets adjustable. These adjustments are over by pressing enter-key without adjustment data save.
V-Linear
With the right/left arrow key, the Vertical Linearity Balance gets adjustable. After the adjustment, the Vertical linearity balance adjustment is over by pressing enter-key without adjustment data save.
Size Data Save
All the menus at the Size menu can be saved.
Auto Size
With the right/left arrow key, select the radio button of AS1 (Saveing the parameter of Size 1), AS2 (Saving the parameter of Size 2 and Position) or AS3 (Saving distortion). After the selection, the Auto Size Save is over by pressing the enter key with the adjustment data saving.

Image Menu
Tilt
With the right/left arrow key, the Tilt gets adjustable. (the up/down key does not work.) After the adjustment, the tilt adjustment is over by pressing enter-key without adjustment data save.
Uniformity
With the right/left arrow key, the landing correction gets adjustable. With the up/down arrow key, however, the adjustment is disable. This adjustment is over by pressing enter-key without adjustment data save.
Convergence
With the right/left arrow key, the horizontal convergence gets adjustable. With the up/down arrow key, the vertical convergence gets adjustable. These adjustments are over by pressing enter-key without adjustment data save.
Moiré Reduction
With the right/left arrow-key the horizontal moiré get adjustable. After the adjustment, the moire adjustment is over by pressing the enter-key without adjustment data save.
V-Conver T/B
With the right/left arrow-key the top vertical convergence get adjustable. With the top/bottom arrow-key, the bottom vertical convergence gets adjustable. After the adjustment, the vertical convergence adjustment is over by pressing the enter-key without adjustment data save.
Top Corners
With the right/left arrow-key the upper left uniformity get adjustable. With the top/bottom arrow-key, the upper right gets adjustable. After the adjustment, the top uniformity adjustment is over by pressing the enter-key without adjustment data save.
Bottom Corners
With the right/left arrow-key the lower left uniformity get adjustable. With the top/bottom arrow-key, the lower right gets adjustable. After the adjustment, the lower uniformity adjustment is over by pressing the enter-key without adjustment data save.
Image Data Save
All the menus at the Image menu can be saved.

Color Menu
Temperature
With the right/left arrow key, the Temperatur gets adjustable. After the adjustment, the temperatur adjustment is over by pressing enter-key without adjustment data save.
Gain
With the right/left arrow key, the R, G, and B Gain and sub-contrast gets adjustable. After the adjustment, the gain adjustment is over by pressing enter-key without adjustment data save.
Cutoff
With the right/left arrow key, the R, G, and B Cut off and sub-contrast gets adjustable. After the adjustment, the cut off adjustment is over by pressing enter-key without adjustment data save.
ABL
With the right/left arrow key, the ABL gets adjustable. After the adjustment, the ABL adjustment is over by pressing enter-key without adjustment data save.
Temp.Param.
With the right/left arrow key, the Tenp.Param gets adjustable. After the adjustment, the Tenp.Param adjustment is over by pressing enter-key without adjustment data save.
However, do not use this icon because this adjustment changes the parameter of the each color temperature.
Color Data Save
All the menus at the Color menu can be saved.
Restoration Save
The Restoration Save is performed.

System
H-Drive
The H-Drive gets adjustable with the right/left arrow key. After the adjustment, the menu returns to system menu by pressing enter-key without saving adjustment data.
H-Size Limit
The H-Size Limit gets adjustable with the right/left arrow key. After the adjustment, the menu returns to system menu by pressing enter-key without saving adjustment data.
V-Size Limit
The V-Size Limit gets adjustable with the right/left arrow key. After the adjustment, the menu returns to system menu by pressing enter-key without saving adjustment data.
H-DF
The H-DF gets adjustable with the right/left arrow key. After the adjustment, the menu returns to system menu by pressing enter-key without saving adjustment data.
V-DF
The V-DF gets adjustable with the right/left arrow key. After the adjustment, the menu returns to system menu by pressing enter-key without saving adjustment data.
H-DF Phase
The H-DF Phase gets adjustable with the right/left arrow key. After the adjustment, the H-DF Phase adjustment is over by pressing enter-key without saving adjustment data.
V-Posi. Limit
The V-Posi Limit gets adjustable with the up/down/right/left arrow key. After the adjustment, the V-Posi Limit adjustment is over by pressing enter-key without saving adjustment data.
System Data Save
All the menus at the System menu can be saved.

Function menus
Setup
The Setup is performed by setting the radio button "OK" with the right/left arrow key and pressing enter key. All the setting gets default with returning Function menu.
Size Delete One
The Size Delete One is performed by setting the radio button "OK" with the right/left arrow key and pressing enter key. The factory size data of which input then is deleted with returning Function menu.
Size Delete All
The Size Delete All is performed by setting the radio button "OK" with the right/left arrow key and pressing enter key. All the factory size data is deleted with returning Function menu.
CopyColor 1-2,3
The Color Copy 1-2,3 is performed by setting the radio button "OK" with the right/left arrow key and pressing enter key. The selected color mode data is copied to another color mode area with returning Function menu.
H-Hold
The H-Hold is performed by setting the radio button "OK" with the right/left arrow key and pressing enter key. After calibration, the Function menu appears.
By setting the radio button "Check" with the right/left arrow key and pressing enter key, the system is set to the free-run and the oscillation frequency of Deflection processing IC is displayed to check the H.Hold. After the check, the Function menu appears.
Contrast Save
The Contrast Save is performed by setting the radio button "OK" with the right/left arrow key and pressing enter key. The contrast and brightness setting data is saved as factory default data with returning Function menu.
Do not use the "Contrast Save" icon not to change the default settings in the Contrast/Brightness.
X Pro setting 
By selecting the "OK", the screen during about 6 seconds becomes black, and the X Pro is adjusted automatically. After the auto adjustment, the adjustment value is displayed under the "XP" icon. Red horizontal lines appear when the X-ray protector has not been adjusted or adjustment error.

Memory menus
Degauss

Select the degauss icon and press the enter key, the "Degauss" is performed with displaying degauss icon. After degaussing, the icon in the main menu changes to Floppy disk icon for approx.1 second. Select this floppy disk icon and press the enter key, you can get into debug mode with the memory menu.

What is the debug mode?

Rewrite the data in the memory IC (EEPROM). On the memory menu, select the address menu with the top/down arrow key. Then, change or set the address with the right/left arrow key. (The address and data is hexadecimal number system. Do not change the address unreaonablly. The set may be destroyed.
