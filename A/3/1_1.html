Sync Sw function

  1. Power On the SW pressing the down key of the Quick Set Control Pad to change the sync mode.

Adjustment Lock

  1. Power On the SW with pressing the AUTO-SIZING Button
  2. To cancel the lock, repeat the above procedure.

How to get into Factory Mode

  1. Display "Reset" in ScreenManager User Mode.
  2. Press the right-arrow key of the Quick Set Control Pad twice for a few seconds to get into the factory mode.


How to return to user mode

After finishing the adjustments at factory mode, be sure to get back into User Mode.

  1. Turn Off the Power SW and turn it ON again. Or, display main menu of the factory mode and select the "EXIT" icon to press the down-arrow key for a few seconds.

How to cancel compulsory factory mode

  1. Turn off the SW after performing SETUP or RESET.
