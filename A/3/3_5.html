Preparation for the Confirmation

Before starting general confirmation, the following tools are recommended to do below tuning

  • Wattmeter
  • Analog signal generator (Recommendation: Astrodesign, Corresponding: VG-823)
  • ROM for adjustment signal
    (ROM Name VG-819 MA-21D0 Ver1.3 )
    *The analog signal generator VG-823 use.
  • Withstanding voltage test tool
  • Low resistance test tool
