Preparation for the Adjustment

Before starting general adjustment, the following tools are recommended to do below tuning

  • Computer       
    • Windows machine : Windows 98, Windows 98 second edition
    • Communication port : USB x 1
  • ROM for adjustment signal
    (ROM Name VG-819 MA-21D0 Ver1.3 )
    *The analog signal generator VG-823 use.
  • High voltage probe
  • Brightness meter
    Contact brightness meter (Recommendation: Minolta CA-100)
  • Scale(more than 400mm)
  • Oscilloscope
  • Digital voltage meter
  • Analog signal generator (Recommendation: Astrodesign, Corresponding: VG-823)
  • High impedance voltage meter
  • Ceramic or plastic flat driver
  • Short clip
  • Rewriting Software   
    • SNEdit190.exe (for rewriting the User time)
