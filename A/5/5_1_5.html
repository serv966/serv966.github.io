Deflection Control Block

SCHEMATIC DIAGRAM for DSP BLK

The Deflection Control Block has DSP LSI (U601) as the core, audio D/A converter (U605, U606) and the horizontal oscillator IC (U602). It generates horizontal and vertical drive wave and the distortion correction wave.

Horizontal Oscillate Circuit (U602)
The U602 outputs the H.DRV pulse from 15 pin from the input sync signals of 30pin and horizontal drive circuit operates. The H.FBP generated in the horizontal output circuit is input to the 8 pin and used to lock the sync.
The DAC (U603) outputs the 29pin voltage for the horizontal position control and the 27 pin voltage for the oscillation frequency control.
Vertical Oscillate Circuit
The U606 (1 pin) outputs the vertical drive wave the DSP LSI operated in the vertical size (amplitude), the vertical position (the DC level) and the vertical linearity.
Horizontal Size Correction Wave
The horizontal size correction wave has four functions; Side-pin, Side-pin corner (Top/Bottom), Side-pin S and Side-pin W. The U606 (4 pin) outputs the horizontal size correction wave the DSP LSI operated and mixed in these functions. The U607 amplifies the wave coming from the U606 to output to the deflection circuit.
Horizontal Phase Correction Wave
The horizontal phase correction wave has the parallelogram, side-pin balance and balance corner (Top/Bottom) functions. The U605 (1 pin) outputs the horizontal phase correction wave the DSP LSI operated and mixed in these functions. This output wave is input to the horizontal position control terminal (29 pin) in the horizontal oscillation circuit (U602).
Vertical Convergence Wave
The U605 (4 pin) outputs the vertical convergence wave generated by the DSP operation. The U608 amplifies this wave to output as V.CONV.
General Output Ports

The DSP LSI (U601) output ports are used as follows;

  • Signal switch (U601, 1pin) of power sync.
  • Freerun control signal (U601, 4pin) of oscillation IC (U602).
Synchronization pulse of power frequency
Pulse (2fH) used for the synchronization pulse of the power supply frequency is generated based on the pulse output from 41pin of DSP LSI.When the horizontal operating frequency is less than 63kHz, the pulse of 2fH synchronization pulse becomes twice as horizontal operating frequency. This switching is done by control signal (U601 1pin).
